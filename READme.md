# Joseph's Movie Watchlist

Joseph's Movie Watchlist is a single-page web application created for
SBA 316: The Document Object Model.

The application allows users to add movies to a personal watchlist,
assign each movie a genre and rating, mark movies as watched, remove
movies, and filter the list based on watch status.

## Features

- Add a movie using a validated form
- Choose a movie genre
- Rate movies from 1 through 10
- Mark movies as watched or unwatched
- Remove movies from the watchlist
- Filter movies by watched status
- Display the total number of movies
- Display a message when the watchlist is empty
- Show alerts and confirmation messages

## Technologies Used

- HTML
- CSS
- JavaScript
- Document Object Model
- Browser Object Model
- Git
- GitHub

## Event Listeners

The application uses the following events:

- `submit`
- `click`
- `change`
- `blur`

## Form Validation

The form includes HTML validation using:

- `required`
- `minlength`
- `maxlength`
- `min`
- `max`

JavaScript validation is also used to check the movie title and rating.

## How to Use the Application

1. Enter a movie title.
2. Select a genre.
3. Enter a rating between 1 and 10.
4. Click the Add Movie button.
5. Mark movies as watched or unwatched.
6. Remove movies from the list.
7. Filter movies by their watch status.
